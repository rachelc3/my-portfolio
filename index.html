Here are some of the projects that I'm most proud of. Click the images to view a live version of the project!

Unplugged

This is a 3-page responsive website for a tech-free retreat
that adapts to mobile, tablet, and desktop screens. The website
includes a spot to put in an email to download an itinerary and
a Google Maps embedded into the About page. HTML, CSS, Flexbox,
and Media Queries helped bring this project to life.

Guess the Word Game

Using JavaScript, players guess the word by entering one
letter at a time. If the player guesses all the letters
correctly before they use up their allotted guesses, they win!

Sticky Notes App

This is an interactive app that will let users create "sticky notes"
on their computer to keep track of to-do items. Using ES6, JSX, and
DOM Manipulation, notes can be edited, added, deleted, and searched
for via the search bar, with all of the notes saved to local storage.
