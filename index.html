Unplugged

Project Description

This is a 3-page responsive website for a tech-free retreat that adapts to mobile, tablet, and desktop screens. The website includes a spot to put in an email to download an itinerary and a Google Maps embedded into the About page. The languages, skills, and tools used to build this project are CSS3, Flexbox, HTML5, Media Queries, Responsive Typography, GitHub, and Text editors.

Guess the Word Game

Project Description

Players guess the word by entering one letter at a time. If the player guesses all the letters correctly before they use up their allotted guesses, they win! JavaScript was one of the main languages used in creating this project.

Sticky Notes App

Project Description

This is an interactive app that will let users create “sticky notes” on their computer to keep track of to-do items. Notes can be edited, added, deleted, and search for via the search bar, with all of the notes saved to local storage. The main skills and coding languages used to build this are ES6, JSX, Managing data flow, and DOM Manipulation.
